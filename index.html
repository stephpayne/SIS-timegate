Timegate

SCORM completion timer overlay — standalone demo. The widget in the corner is live. Interact with the page to reset the idle timer.

State 1
Counting
Timer is active. The widget shows time remaining and counts down while the learner is on-page and not idle.
State 2
Paused — Idle
No activity detected for the idle timeout window. Timer stops and the widget pulses orange until activity resumes.
State 3
Time Requirement Met
Minimum time has elapsed. Widget turns green and prompts the learner to complete remaining content before exiting.

Active Configuration

Note: In production, these values come from timegate.config.json inside the SCORM package. This demo uses window.TIMEGATE_CONFIG to inject them directly. SCORM completion gating is disabled here — no LMS is present.
